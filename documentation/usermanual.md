# User Manual

## Overview

Skill Swap Marketplace helps students post listings and connect with other users.
You can create an account browse listings create your own posts and manage your profile.

## Create an Account

1. Open the website.
2. Select **Sign up**.
3. Enter your display name email and password.
4. Select **Create account**.

If you already have an account select **Login** instead.

## Log In

1. Open the **Login** page.
2. Enter your email and password.
3. Select **Login**.

After login you will go to the **Skill Swap** page.

## Browse Listings

1. Open the **Skill Swap** page.
2. Look through the listing cards.
3. Use **Category** **Type** and **Search** to narrow the results.
4. Select a user name if you want to open that user profile.

## Create a Listing

1. Open the **Skill Swap** page.
2. Select **Create listing**.
3. Choose a listing type.
4. Choose a category.
5. Enter a title and description.
6. Select **Post listing**.

For a course listing:

1. Choose **Course** as the category.
2. Type a course code.
3. Pick the correct section from the results.
4. Select **Post listing**.

## Show Interest in a Listing

1. Open the **Skill Swap** page.
2. Find a listing from another user.
3. Select **I'm Interested**.
4. The contact email will appear on the listing card.

If you change your mind select **Remove interest**.

## Manage Your Listings

1. Open the **Dashboard** page.
2. Review your listings.
3. Select **Edit** to update a listing.
4. Select **Close listing** when the listing is no longer active.
5. Select **Reopen** if you want to make it active again.
6. Select **Show interested users** to view users who responded.

## Update Your Profile

1. Select your display name in the top bar.
2. Go to the **Skills** section.
3. Type a skill or choose one from the suggestions.
4. Select **Add** to save it.
5. Use the remove button on a skill if you want to delete it.

## Change the Theme

1. Select the theme button in the top bar.
2. The site will switch between light mode and dark mode.

## Simple Tips

1. Use a clear title.
2. Add enough detail in the description.
3. For course listings choose the correct section before posting.
4. Check **Dashboard** often to manage your listings.
